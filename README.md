## Development

Requirements:
- Node.js 18.x (project declares this in `package.json`)

Install dependencies and run local dev server (PowerShell):

```powershell
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```powershell
npm run build
npm run preview
```
